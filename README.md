ChatForge is a ready-to-use backend component designed specifically for production-grade chat applications. It solves a critical problem: developers shouldn't have to reinvent the wheel every time they build a chat application. At the same time, ChatForge recognizes that every application has different needs—different business domains, deployment environments, and capabilities. To balance these requirements, ChatForge uses a hexagonal architecture pattern with ports and adapters, allowing you to customize and extend the framework while keeping the core logic clean and reusable.

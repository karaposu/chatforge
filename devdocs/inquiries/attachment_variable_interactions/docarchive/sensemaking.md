### Ambiguity 2 — How does signal specificity fit into the variable-magnitude structure?

**Resolution: SPECIFICITY IS A MAGNITUDE FACTOR WITHIN EACH VARIABLE.**

Formulation: `effective_magnitude(variable) = nominal_content(variable) × specificity(signal)`. Generic signals have low specificity → low effective magnitude. Specific signals have high specificity → high effective magnitude.

**Counter:** Could specificity be a separate 5th attachment variable?
- Test: can you have high specificity with zero nominal content? No — specificity requires content to be specific ABOUT. They're coupled.
- Specificity is a QUALITY OF a signal, not a separate signal. Better as magnitude-factor than new variable.

**Why the counter fails:** A 5th variable would need to satisfy the qualification criteria (multiplicative on attachment, dual character, orthogonal to existing variables, etc.). Specificity has no independent contribution apart from its role in scaling existing variables' effective magnitude. It's a quality factor, not an independent source.

**What's fixed:** Signal specificity is a magnitude modifier per variable, not a separate attachment variable.

**What's no longer allowed:** Treating specificity as a 5th attachment variable.

**Depends on:** Operationalization of signal detection must distinguish generic vs specific within each variable.

**Changed:** Makes explicit what was implicit.

### Ambiguity 3 — How does sender-side SP-reading work in single-message contexts?

In multi-interaction contexts, modulator signals can be observed longitudinally (especially Coherence). In single-message contexts (cold email, Reddit DM, LinkedIn first contact), the receiver has only the message to work with. The message's STYLE carries the sender's modulator signal:

- Generic template → Supplication display (low sender-SP)
- Specific engaged message → Self-Focus display (high sender-SP)
- Contradictory claims / misaligned signals → Coherence signal (low sender-g₂)
- Emotional dysregulation visible → Emotional Composure signal (low sender-g₃)

The receiver's attachment to the sender is modulated by these single-message-readable modulator signals.

**Counter:** Could single-message modulator-reading be unreliable and shouldn't be relied on in the theory?
- Test: empirically, people DO form attachment impressions from single messages. The mechanism works; its accuracy is a separate question.
- The theory describes mechanism; accuracy is a downstream empirical concern.

**What's fixed:** Message style IS the sender's modulator signal in single-interaction contexts. The mechanism is the same as iteration-3.2's; just explicit for the single-message case.

**What's no longer allowed:** Claiming modulator reading requires longitudinal observation (Coherence does; Self-Positioning doesn't).

**Depends on:** PRAGMA → sender-modulator mapping for single-message detection. Operationalization of message-style-as-modulator-signal.

**Changed:** Mechanism made explicit for commonly-occurring case (first-contact outreach).

### Ambiguity 4 — Does attachment have multiple output dimensions?

**Resolution: YES — ATTACHMENT HAS MAGNITUDE AND TYPE.**

- **Magnitude:** scalar quantity from f-sum × g-product. "How strongly does the observer engage?"
- **Type:** qualitative property from variable mix inside f. "What KIND of attachment is this?"

Illustrative TYPEs:
- Charm-dominant → status / admiration attachment
- Hope-dominant → transactional attachment
- Fear-dominant → coerced attachment
- Resonance-dominant → bonded / deep attachment
- Mixed → hybrid types with qualitative character blended

**Counter:** Could TYPE be reducible to MAGNITUDE (i.e., what seems like different types is actually different magnitudes)?
- Test: two attachments of equal magnitude but different variable mixes — do they behave differently? Yes: Fear-dominant attachments evaporate when fear is removed; Resonance-dominant persist despite external changes. Same magnitude, different temporal dynamics → different TYPE.
- TYPE is a real distinct dimension.

**Why the counter fails:** Magnitude captures instantaneous engagement strength. TYPE captures persistence dynamics and qualitative character over time. Different dimensions; one doesn't reduce to the other.

**What's fixed:** Attachment has two output dimensions. Spec should document both.

**What's no longer allowed:** Treating attachment as purely scalar (magnitude only) when variable mix carries qualitative information.

**Depends on:** APT Profiling output can report attachment TYPE alongside MAGNITUDE.

**Changed:** Clarifies that f-sum isn't the only output of f; f's variable mix also carries information.

Note: Illustrative TYPE taxonomy (transactional, bonded, status-driven, coerced, hybrid) is provisional. Comprehensive TYPE classification may need downstream empirical work. iteration-3.2.1 commits to the dimension existing; specific taxonomy is open.
